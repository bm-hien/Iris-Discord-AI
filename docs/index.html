üöÄ Quick Start

1

Clone Repository

git clone https://github.com/bm-hien/Iris-Discord-AI.git
cd Iris-Discord-AI
2

Install Dependencies

npm install
3

Configure Environment

cp .env.example .env
# Edit .env with your Discord bot token
4

Start the Bot

npm start

Requirements

  • Node.js 16+ required
  • Discord Bot Token
  • AI Provider API Keys (optional)

Free API Keys

  • Gemini API - 15 req/min
  • Groq API - Fast responses

üîó API Reference

Core Functions

generateResponse()

Main AI response generation function

Parameters:
  • message - Discord message object
  • context - Conversation context
  • options - Additional options
imageToBase64()

Convert image URL to base64 for AI processing

Parameters:
  • imageUrl - URL of the image
extractUrls()

Extract URLs from text messages

Parameters:
  • text - Text to extract URLs from

Database Functions

setUserApiKey()

Store user's personal API key

clearConversationHistory()

Clear user's conversation history

getCustomSystemMessage()

Get user's custom AI settings
